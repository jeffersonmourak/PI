π

O π é um player interativo baseado no video element do HTML5 e no polyfill Polymer da Google, a ideia principal do π é oferecer todos os recursos do embed do YouTube mas te deixar livre para estilizar seu player como quiser.
com ele você pode fazer muitas interações com os vídeos.

Assitir os vídeos diretamente do youtube sem a necessidade de utilizar o player do youtube, basta adicionar o elemento "youtube" com o id do vídeo e pronto, o vídeo será carredado diretamente do youtbe

Carregar vídeos locais em MP4 e WEBM

Adicionar anotações durante o vídeo

Além de deixar poder deixar o player com a sua cara, alterando apenas um arquivo.

Como functiona?

Para a dicionar o player a sua página basta baixar o projeto no GitHub, depois de baixar você deve importar os arquivos do Polymer ao seu site, usando: 

<script src="bower_components/platform/platform.js"></script>
<link rel="import" href="bower_components/polymer/polymer.html">
depois deve ser importado o gerador do player 
<link rel="import" href="pi.html">

e por fim criar seu player

<pi-player> </pi-player>

Usando as funcionalidades

Pegando um vídeo do YouTube

para importar um vídeo do youtube você só precisa do video ID que fica na barra de endereços depois de 
http://youtube.com/watch?v=
esse valor vai ser adicionado no atributo 
youtube
do π assim:
<pi-player youtube="videoID"> </pi-player>

Pegando um vídeo do meu servidor

O π atualmente suporta os arquivos no formato MP4 e WEBM, para adiciona-los ao player é assim: 

<pi-player mpFourVideo="path/of/video.mp4" webmVideo="path/of/video.webm"> </pi-player>
e pronto!

Adicionando uma anotação no vídeo.

Para se adicionar uma anotação você usa o JavaScript com o objeto pi. veja o exemplo 

<script>
		
pi.setAnnotation("seletorCSS",{

	start : "0:4", //tempo inicial

	end : "0:7", //tempo final

	parameters : {

		text : "Ops, Não é no YouTube Não", //texto da anotação

		width : "250px", //largura da anotação

		height : "auto", //altura da anotação

		top : "164.5px", //distancia do topo

		left : "183.5px", //distancia da esquerda

		style : {

			background : "rgba(255,0,0,0.7)",//cor do plano de fundo

			foreColor : "rgba(255,255,255,1)" //cor do texto

		}

	}});

</script>
